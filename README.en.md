# Codex Skin Marketplace

This is the Codex Skin marketplace maintained by 终端极客. It currently includes:

- `codex-skin`: installs the `salary-cat` / 月薪喵 theme by default, with theme switching, skin creation, theme export, and native restore support.

## Install

Clone this marketplace repository:

```bash
git clone https://github.com/mcgfdata/codex-skin-marketplace.git
```

Add it to Codex:

```bash
codex plugin marketplace add /path/to/codex-skin-marketplace
```

Then ask Codex in a new window:

```text
帮我设置codex皮肤 mcgfdata/codex-skin ，作者是 终端极客
```

or:

```text
帮我安装codex-skin，作者是：终端极客
```

Codex should match the `codex-skin` plugin and install the 月薪喵 theme by default.
